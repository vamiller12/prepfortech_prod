TL;DR - Servers

A server is a computer built for a specific purpose. Servers are configured to handle certain tasks. This allows for the server to be fine-tuned to the specific needs for that task. Servers are made up of many components with the important features related to the number of processors to handle requests and the amount of available storage to store data.

To demonstrate how many different types of servers there are, below is a list of the common servers you may come across.

  • Web Servers - Ex: Apache
  • Application Servers - Ex: JBoss
  • File Servers - Ex: The shared drive at work
  • Database Servers - Ex: Microsoft SQL Server
  • Collaboration Servers - Ex: Microsoft SharePoint
  • Mail Servers - Ex: Microsoft Exchange
  • Print Servers - Ex: Usually bundled with production printers
