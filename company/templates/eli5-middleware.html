TL;DR - Middleware

A piece of software that acts as the glue between operating systems and other applications. Just as the operating system works to manage the hardware, the middleware is what allows the operating system to function with the applications. The middleware is application and operating system specific which is why some programs will not run on certain operating systems. Remember the last time you wanted to connect to a new printer but you had to install the driver first? That driver is a piece of a middleware that allows you to interact with the printer.
