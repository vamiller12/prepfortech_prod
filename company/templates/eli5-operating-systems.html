TL;DR - Operating Systems

Software that sits between the hardware and applications on a computer or machine and manages the interaction between both. The operating system does basic system control allowing the end use to interact with the user interface and not have any knowledge of how the system is working. Taking an action such as launching a program takes strict coordination for the application code to be able to connect with the machine components and behave reliably.
