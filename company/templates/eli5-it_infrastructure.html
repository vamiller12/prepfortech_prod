IT Infrastructure

TL;DR

IT Infrastructure is the broad category of all hardware, software, and network components needed for a company to perform their day-to-day operations. The hardware and software chosen is dependent on several factors that include both technical and personal preferences as well as cost.

Hardware Overview:

The physical (or virtualized) hardware choices center around storage, servers, switches, routers, network devices, and personal computers. Each hardware piece within an infrastructure setting has a defined purposes. Those purposes are:

  • Storage: The physical place that holds the data. It is common for the storage devices to be located on-premises or in data centers.
  • Servers: A combination of hardware and software components that are configured to handle specific tasks.
  • Switches: Hardware that facilitates wired connection between other devices allowing the devices to communicate.
  • Routers: Hardware that directs internet traffic.
  • Computers/Mobile Devices: The typical devices employed by end users to connect to networks.
    • Software Overview:

    The software choices include operating systems, middleware, security systems, and frontend systems. Each software piece is responsible for interacting with the software for management, setting has a defined purposes. Those purposes are:

  • Operating System: Software that manages the hardware, applications, and operations of piece of hardware.
  • Middleware: Software that connects specific applications to operating systems, allowing the application to run on the operating system
  • Containers: Virtualization software create separate environments on expense hardware devices
  • Security Systems: Applications that are designed to keep hardware and software safe from unwanted visitors and malicious activities.
